# 招商银行信用卡账单格式参考

## 账单结构特征

### 常见账单格式类型
1. PDF格式电子账单
2. 短信通知格式
3. APP推送账单摘要
4. 邮箱账单

### 关键字段识别
- 卡号后四位：通常显示为 "*** **** **** 4336" 格式
- 账单周期：如 "2025-12-01 至 2025-12-31"
- 交易日期：YYYY-MM-DD 格式
- 记账日期：YYYY-MM-DD 格式
- 交易金额：带人民币符号 ¥ 或 CNY 标识
- 商户名称：交易对方的商户名

### 招商银行常见商户分类
- 餐饮类：如 "支付宝-肯德基", "美团外卖"
- 购物类：如 "天猫超市", "京东商城"
- 交通类：如 "滴滴出行", "高德打车"
- 生活缴费：如 "电费", "燃气费", "话费充值"
- 娱乐休闲：如 "猫眼电影", "腾讯视频"

## 数据提取要点

### 金额识别规则
- 正数表示支出，负数表示退款或收入
- 保留两位小数
- 注意区分人民币和其他币种

### 时间范围确定
- 根据账单周期确定分析范围
- 对于月度报告，通常包含整月所有交易
- 考虑时区差异（中国标准时间UTC+8）

### 消费分类算法
1. 关键词匹配法：通过商户名称关键词自动分类
2. 金额区间法：根据消费金额大小判断用途
3. 时间规律法：根据消费时间频率判断类别
